# Neptuno Managing Server

https://neptuno-managing.herokuapp.com/

## Getting Started

### Prerequisites

Nodejs >= 8.3.0
Mongodb >= 3.6

### Development

```sh
npm install
```

```sh
npm run db:init
```

```sh
npm run dev
```

### Testing

Testing by module. Example

```sh
npm run test:utils
```

Run all test

```sh
npm run test
```
